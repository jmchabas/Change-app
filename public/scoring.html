LifeOS Scoring Guide

How your execution score is calculated (0-100). Mood is tracked separately.

Weekday: fixed denominator Weekend: optional fields are dynamic

Weekdays (Mon-Fri)

Execution score = (behavior points / 80) × 100

  • Behavior points: up to 80 across Focus, Eating, Clean Eve, Work, Personal, Gym, Kids, Bedtime (10 each).
  • Mood is tracked separately (1-10) and not included in the execution score.

Weekends (Sat-Sun)

Work win, Personal win, and Bedtime are optional.

Execution score = (behavior points / active behavior points) × 100

  • Active behavior points = 50 (5 required habits) + optional habits that are filled.
  • Blank or invalid optional inputs are excluded from the denominator.

This keeps weekends fair while still rewarding optional completions.

Scoring Bands

Overall daily score bands

  • 80-100: strong day (green zone)
  • 60-79: mixed day (amber zone)
  • 0-59: recovery needed (red zone)

Metric-level bands

  • Escape media minutes: 0m = 10, <=15m = 8, <=30m = 6, <=60m = 3, >60m = 0
  • Meals outside window: 0 = 10, 1 = 6, 2 = 3, >=3 = 0
  • Bedtime (target 9:30pm): on/before = 10, <=30m late = 8, <=60m late = 5, <=90m late = 2, later = 0
  • Mood: 1-10, tracked as a separate wellbeing metric (not in execution score)
  • Binary habits (Clean Eve, Work, Personal, Gym, Kids): Yes = 10, No = 0
  • Sleep card colors: Sleep (green >= 7h, amber >= 6h, red < 6h) · Sleep score (green >= 80, amber >= 65, red < 65)

How ✓ / ✗ are computed in habit charts and grid

  • Focus: ✓ when focus points are >= 6, else ✗
  • Eating: ✓ when eating points are >= 6, else ✗
  • Bed: ✓ when bedtime points are >= 8, else ✗
  • All other habit rows are direct Yes/No values from your check-in.

Notes

  • Bedtime target is 9:30pm.
  • Form values are persisted even when weekend optional fields are left blank.
  • Trend arrows in the morning brief compare vs previous day.
